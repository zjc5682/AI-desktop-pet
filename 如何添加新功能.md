## 2️⃣ 添加新动作类型

### 场景：想让角色有"开心"的动作

### 📍 步骤 1：准备动作文件

确保 `model.json` 中有这个动作：

```json
{
  "motions": {
    "happy": [  // ← 新增的动作类型
      {"file": "motions/happy_01.mtn"},
      {"file": "motions/happy_02.mtn"}
    ]
  }
}
```

### 📍 步骤 2：在代码中调用

**方法 A：在现有逻辑中添加**

编辑 `src/components/Live2DPet.vue`，在合适的地方添加：

```typescript
// 例如：在点击头部时，除了原有动作，还播放开心动画
const handleClick = (e: MouseEvent) => {
  if (isInHitArea(x, y, currentModel.value.hitAreas.head)) {
    canvasRef.value.playMotion('flick_head');  // 原有动作
    canvasRef.value.playMotion('happy');       // ← 新增：同时播放开心动作
    messageState.showRandom(messageState.messages.click.head);
  }
};
```

**方法 B：创建新的触发方式**

例如：双击时播放开心动作

```typescript
// 在 Live2DPet.vue 中添加双击监听
const handleDoubleClick = (e: MouseEvent) => {
  if (!model || isSleeping.value) return;
  
  const rect = canvasRef.value.canvas.getBoundingClientRect();
  const { x, y } = getNormalizedCoords(e.clientX, e.clientY, rect);
  
  if (isInHitArea(x, y, currentModel.value.hitAreas.head)) {
    canvasRef.value.playMotion('happy');  // 播放开心动作
    messageState.show('好开心呀~');        // 显示消息
  }
};

// 在 onMounted 中添加监听器
onMounted(async () => {
  await setupTauriWindow();
  window.addEventListener('mousemove', handleMouseMove);
  window.addEventListener('click', handleClick);
  window.addEventListener('dblclick', handleDoubleClick);  // ← 新增
});

// 在 onUnmounted 中移除监听器
onUnmounted(() => {
  window.removeEventListener('mousemove', handleMouseMove);
  window.removeEventListener('click', handleClick);
  window.removeEventListener('dblclick', handleDoubleClick);  // ← 新增
});
```

## 3️⃣ 添加新交互功能

### 场景：鼠标滚轮控制角色大小

### 📍 步骤 1：添加状态变量

在 `Live2DPet.vue` 中：

```typescript
// 在现有的 ref 定义后面添加
const currentScale = ref(1.0);  // 当前缩放比例
```

### 📍 步骤 2：添加滚轮事件处理

```typescript
// 添加滚轮处理函数
const handleWheel = (e: WheelEvent) => {
  if (!canvasRef.value) return;
  
  // 根据滚轮方向调整缩放
  const delta = e.deltaY > 0 ? -0.1 : 0.1;
  currentScale.value = Math.max(0.5, Math.min(2.0, currentScale.value + delta));
  
  // 通知 Canvas 组件调整大小
  // 需要通过某种方式传递给 Canvas（见步骤 3）
};
```

### 📍 步骤 3：传递数据给子组件

**修改 Live2DCanvas.vue**：

```vue
<script setup lang="ts">
interface Props {
  modelPath: string;
  width: number;
  height: number;
  isSleeping: boolean;
  customScale?: number;  // ← 新增：自定义缩放
}

// 监听缩放变化
watch(() => props.customScale, (newScale) => {
  if (newScale && model) {
    model.scale.set(newScale);
  }
});
</script>
```

**修改 Live2DPet.vue**：

```vue
<template>
  <Live2DCanvas
    ref="canvasRef"
    :model-path="currentModel.path"
    :width="FIXED_WIDTH"
    :height="FIXED_HEIGHT"
    :is-sleeping="isSleeping"
    :custom-scale="currentScale"  <!-- ← 传递缩放值 -->
    @model-loaded="handleModelLoaded"
    @motion-played="handleMotionPlayed"
  />
</template>
```

### 📍 步骤 4：添加事件监听

```typescript
onMounted(async () => {
  await setupTauriWindow();
  window.addEventListener('mousemove', handleMouseMove);
  window.addEventListener('click', handleClick);
  window.addEventListener('wheel', handleWheel);  // ← 新增
});

onUnmounted(() => {
  window.removeEventListener('mousemove', handleMouseMove);
  window.removeEventListener('click', handleClick);
  window.removeEventListener('wheel', handleWheel);  // ← 新增
});
```

**完成！** 现在滚动鼠标滚轮就可以调整角色大小了 🎉

## 5️⃣ 添加新组件

### 场景：添加一个显示时间的组件

### 📍 步骤 1：创建新组件

新建文件 `src/components/Live2DPetComponents/TimeDisplay.vue`：

```vue
<!--
  TimeDisplay.vue - 时间显示组件
  功能：在屏幕角落显示当前时间
-->
<template>
  <div class="time-display">
    {{ currentTime }}
  </div>
</template>

<script setup lang="ts">
import { ref, onMounted, onUnmounted } from 'vue';

const currentTime = ref('');

// 更新时间
const updateTime = () => {
  const now = new Date();
  currentTime.value = now.toLocaleTimeString('zh-CN', {
    hour: '2-digit',
    minute: '2-digit'
  });
};

let timer: number | null = null;

onMounted(() => {
  updateTime();  // 立即更新一次
  timer = window.setInterval(updateTime, 1000);  // 每秒更新
});

onUnmounted(() => {
  if (timer) clearInterval(timer);  // 清理定时器
});
</script>

<style scoped>
.time-display {
  position: absolute;
  top: 10px;
  right: 40px;  /* 在控制面板左边 */
  background: rgba(255, 255, 255, 0.9);
  padding: 8px 12px;
  border-radius: 8px;
  font-size: 12px;
  color: #333;
  z-index: 10001;
}
</style>
```

### 📍 步骤 2：在主组件中使用

编辑 `src/components/Live2DPet.vue`：

```vue
<template>
  <div class="pet-container" :style="containerStyle">
    <Live2DCanvas ... />
    <Live2DMessage ... />
    <ControlPanel ... />
    
    <!-- 新增时间组件 -->
    <TimeDisplay />
  </div>
</template>

<script setup lang="ts">
// 导入新组件
import TimeDisplay from './Live2DPetComponents/TimeDisplay.vue';
</script>
```

**完成！** 现在右上角会显示当前时间 ⏰

### 案例 1：添加随机眨眼功能

```typescript
// 在 Live2DCanvas.vue 中添加
let blinkTimer: number | null = null;

const blink = () => {
  if (!model) return;
  
  // 闭眼
  model.model.setParameterValueById('ParamEyeOpenL', 0);
  model.model.setParameterValueById('ParamEyeOpenR', 0);
  
  // 0.1 秒后睁眼
  setTimeout(() => {
    model.model.setParameterValueById('ParamEyeOpenL', 1);
    model.model.setParameterValueById('ParamEyeOpenR', 1);
  }, 100);
  
  // 3-6 秒后再次眨眼
  const nextBlink = 3000 + Math.random() * 3000;
  blinkTimer = window.setTimeout(blink, nextBlink);
};

onMounted(() => {
  // ... 模型加载后
  blink();  // 启动眨眼循环
});

onUnmounted(() => {
  if (blinkTimer) clearTimeout(blinkTimer);
});
```

## 🛡️ 开发最佳实践

### ✅ 推荐做法

1. **小步测试**：每次只改一点，测试是否正常
2. **查看控制台**：F12 打开开发者工具看错误信息
3. **利用热更新**：Vue 组件修改后自动刷新
4. **备份原文件**：大改动前先复制一份

### ❌ 避免的做法

1. **一次性改太多**：出问题难以定位
2. **不看错误信息**：控制台是最好的老师
3. **忘记清理资源**：定时器、事件监听器要记得移除
