## 🔧 常见问题解决

### ❌ 模型加载失败

**检查清单**：
- [ ] `model.json` 路径是否正确
- [ ] `.moc` 文件是否存在
- [ ] 贴图路径是否在 `model.json` 中正确配置
- [ ] 浏览器控制台是否有错误信息

---

### ❌ 视线不跟随鼠标

**调试方法**：
1. 打开浏览器控制台（F12）
2. 查找 `🔍 Live2D 模型调试信息` 日志
3. 检查输出中的方法和参数列表
4. 确认 `setParameterValueById` 方法存在

---

### ❌ 点击没反应

**检查**：
```typescript
// 在 Live2DPet.vue 中查看 hitAreas 配置
hitAreas: {
  head: { x: [-0.8, 0.8], y: [0.3, -0.5] }  // ← 坐标范围是否合理
}
```

**调整方法**：修改坐标值，使其匹配角色的实际位置

---

### ❌ 窗口不是透明的

**检查**：
1. `App.vue` 中背景是否为透明
2. `Live2DCanvas.vue` 中 `backgroundAlpha: 0`
3. Tauri 配置中 `transparent: true`

## 📊 性能优化建议

### ✅ 推荐做法
- 使用 `.min.js` 压缩版库文件
- 贴图使用合适的分辨率（不要过大）
- 及时清理定时器（composables 已处理）

### ❌ 避免做法
- 不要频繁创建/销毁组件
- 不要在循环中调用 `show()`
- 不要移除必要的 `onUnmounted` 清理代码
